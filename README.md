ElJasus is arabian muliplayer game built for family and friends

1- ***Gameplay***
- The game is simple, up to 10 players, 1 is spy, all see the secret word(food or anything else selected by category) but the spy doesn't
- The innocents doesn't know who's the spy and all what they need to do is to ask each other questions about the secret word specs, and because the spy doesn't know the word he might say something wrong
- If the innocents voted the right spy the spy still have a chance to guess the secret word using his information he collected from the hints or questions and answers of innocents and if he chosen the right word, he wins
- If the vote ended by draw the game makes random select
- There is text chat and voice chat in the room

2- ***Security***
- The text chat has a high moderation system that gives 1 warning before the ban
- The ban works on the account and on the device to prevent signing in with another account with same device

3- ***features***
- Friending system
- Reporting system
- Rooming system
- Ranking system
- Coins system
- Announcement system
